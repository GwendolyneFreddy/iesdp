--- n: 365 name: "Make unselectable" param1: "Dialogue enabled" param2: "AI enabled" special: "Selection circle color" bg1: 0 bg2: 0 bgee: 1 iwd1: 0 iwd2: 0 pst: 0 --- This effects makes the targeted creature unselectable.
Known values for Dialogue Enabled are:
BG:EE/SoD:
  • 0 ⟶ Enabled
  • 1 ⟶ Disabled
BG2:EE:
  • 0 ⟶ Disabled
  • 1 ⟶ Enabled

Known values for AI Enabled are:
  • 0 ⟶ Disabled
  • 1 ⟶ Enabled

Known values for Selection circle color are:
  • 0 ⟶ Purple Selection Circle
  • non-zero ⟶ Normal Selection Circle
{% capture note %} AI Enabled doesn't work in BG:EE/SoD games (i.e., you cannot disable the AI of the targeted creature). {% endcapture %} {% include note.html %}
