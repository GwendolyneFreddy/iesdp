--- n: 323 name: "Stat: Turn Undead Level" param1: "Statistic Modifier" param2: "Type" bg1: 0 bg2: 0 bgee: 1 iwd1: 0 iwd2: 0 pst: 0 --- Applies the modifier value specified by the Statistic Modifier field in the style specified by the Type field.
Known values for Type are:
  • 0 ⟶ Cumulative Modifier: TurnLevel = TurnLevel + 'Statistic Modifier' value
  • 1 ⟶ Flat Value Modifier: TurnLevel = 'Statistic Modifier' value
  • 2 ⟶ Percentage Modifier: TurnLevel = (TurnLevel * 'Statistic Modifier' value) / 100
{% capture note %} Only Clerics and Paladins will display their Turn Undead level on the Record Screen. {% endcapture %} {% include note.html %}
