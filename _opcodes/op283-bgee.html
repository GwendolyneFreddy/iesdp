--- n: 283 name: "Use EFF File (Cursed)" param1: "IDS Entry" param2: "IDS File" bg1: 0 bg2: 0 bgee: 1 iwd1: 0 iwd2: 0 pst: 0 --- Applies the eff file specified by the resource key to the targeted creature(s), if the targeted creature(s) match the IDS Entry in the specified IDS File. The eff can be removed via the Remove Curse opcode (in addition to other standard methods).
Known values for IDS File are:
  • 2 ⟶ EA.IDS
  • 3 ⟶ GENERAL.IDS
  • 4 ⟶ RACE.IDS
  • 5 ⟶ CLASS.IDS
  • 6 ⟶ SPECIFIC.IDS
  • 7 ⟶ GENDER.IDS
  • 8 ⟶ ALIGN.IDS
  • 9 ⟶ KIT.IDS
