--- n: 309 name: "Script: Modify Local Variable" param1: "Value" param2: "Type" bg1: 0 bg2: 1 bgee: 1 iwd1: 0 iwd2: 0 pst: 0 --- Set both the local variable specified by the resource key to the value specified by the Value field.
Known values for Type are:
  • 0 ⟶ Set Local (to Value)
  • 1 ⟶ Increment Local (by Value)
{% capture note %} Since the resource key can only be 8 characters long maximum, you can not set a variable with this Effect that has a name longer than this amount.
You can use the VVC field of external EFF files for an additional 8 characters of variable name.
You can use the 3rd resource field of EFF files for an additional 8 characters. {% endcapture %} {% include info.html %} {% capture note %} Value can store integers or coordinates. Coordinates are stored as first 2 bytes X, last 2 bytes Y (f.i.: 310 + (30 << 16)). {% endcapture %} {% include info.html %}
