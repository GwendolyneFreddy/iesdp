--- n: 191 name: "Spell: Casting Level Modifier" param1: "Modifier" param2: "Type" bg1: 0 bg2: 0 bgee: 1 iwd1: 0 iwd2: 0 pst: 0 --- Applies the value specified by the Modifier field as a bonus to casting spell levels, of a type specified by the Type field.
Known values for Type are:
  • 0 ⟶ Mage
  • 1 ⟶ Cleric

This effect modifies CASTINGLEVELBONUSMAGE or CASTINGLEVELBONUSCLERIC depending on Type.
