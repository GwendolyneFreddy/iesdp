--- n: 329 name: "Spell Effect: Slow Poison" param1: "Slow Factor" param2: "Irrelevant" bg1: 0 bg2: 0 bgee: 1 iwd1: 0 iwd2: 0 pst: 0 --- Applies the slow poison effect to the targeted creature(s), using the Slow Factor field. {% capture note %} This opcode writes over the Frequency Multiplier field of any opcode #25 effects on the targeted creature. As a result, it is useless against opcode #25 whose Type is 0 or 1, as well as any Poison effect that is used through opcode #177, as Type=4 requires. {% endcapture %} {% include note.html %} {% capture note %} It can also accelerate Poisons if their Frequency Multiplier field was already higher than the value specified in opcode #329. {% endcapture %} {% include note.html %} {% capture note %} It does not alter the duration of the poison, nor does it affect any of the other frequency opcodes (78/98/272). {% endcapture %} {% include note.html %}
