BG(2)EE Identifiers

Correlates a general time of day, to a specific hour.
Used by the following script triggers:
  • Time()
  • TimeGT()
  • TimeLT()
Used by the DayNight() action.
As can be seen from the file, midnight is defined as 0, and time builds from there, wrapping from 23 back to 0.
The file is extensible, and new entries can be added for example, adding 4 MYTIME will allow MYTIME to be used in scripts, giving the time 3am.
