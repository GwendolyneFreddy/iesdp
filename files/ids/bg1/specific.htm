BG1: TotS Identifiers

Relates to offset 0x273 in the .cre file format.
Used by the Specifics(O:Object*,I:Specifics*Specific) trigger and the ChangeSpecifics(O:Object*,I:Value*Specific) action.
The file is extensible, and new entries can be added for example, adding 55 MYSPECIFIC will allow MYSPECIFIC to be used in scripts.
