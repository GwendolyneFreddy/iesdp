scrtxt01.2da

Defines the textbox shown at each chapter.
First name (in our example ch04ashp) determines BMP reference for text screen.
The first column is filled only by identifiers.
The second column lists the title of the textbox screen.
The third column lists the main text of the textbox screen.
In BG2 ToB we have (scrtxt01.2da):
2DA V1.0
ch04ashp
           0          1          
SWITCH     DEFAULT    DEFAULT    
DEFAULT    8632       55714
