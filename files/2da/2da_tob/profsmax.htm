profsmax.2da

This file is ignored in BG2: ToB
This file defines how many proficiency points a character can be assigned at his/her creation and later levels.
First column is the list of classes identifiers
2nd column is the maximum number of proficiency points given at the PC creation from the class defined in column 1st.
3rd column is the maximum number of proficiency points that can be put in weapon proficiencies on other levels.
