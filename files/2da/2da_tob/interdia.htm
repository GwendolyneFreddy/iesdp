interdia.2da

This file is used to assign dialogue files to PNPCs in game . The dialogues consist of interactions with main character or between party members.
The first column indicates the name of NPC (death variable).
The second one assigns the DLG file with needed dialogues.
The third one is available only in ToB - assigns DLG file for NPC in ToB.
