dexmod.2da

This file defines the influence of dexterity on characters actions, such as dodging missiles and AC bonuses.
The first row (titles of columns ) is filled by the names of actions.
The first column is filled by value of dexterity of character.
The other columns are filled by values for given dexterity.
The reaction value is unused, but it is displayed in the character sheet.
