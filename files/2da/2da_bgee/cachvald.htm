cachvald.2da

This file lists the file-sizes of BIFF files, and is used to determine if a BIFF will fit within the allocated cache size. The file is likely unused in BG(2)EE (the values in the file are hold-overs from BG1).
The first column is unused.
The second column holds the names of the BIFF files.
The third column lists the minimum size required in the cache to hold a copy of the BIFF file.
