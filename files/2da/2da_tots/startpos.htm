startpos.2da

This file assigns the position for characters in party at the beginning of specified part of game.
The first column is filled by the identifiers of selected position feature. This indicates also what part of game uses these values like tutorial or normalgame.
First row contains numbers that indicate the number of party member.
Other rows contain of numbers that are coordinates described by identifier in first column.
