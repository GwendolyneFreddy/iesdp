arealink.2da

This file is not fully understood, but it is thought to control caching of biff files.

The first column the filenames of biff files.
The second column lists the number of valid entries (e.g. 2 means consult the AREARESREF1 and AREARESREF2 columns).
Subsequent columns list biff files to cache the the initial biff is loaded
