intmod.2da

This file defines intelligence influence on characters abilities like success in spell learning or max possible maximal spells level that can be obtained by creature.
The first row (titles of columns ) is filled by the names of features affecting of PC (like LEARN_SPELL).
The first column is filled by value of intelligence of character.
The second column is the chance of successfully learning a spell.
The third column is maximum spell level available.
The fourth column is maximum number of spells per level.
The fifth column is control bonus proficiencies.
