clab****.2da

This file defines what special abilities/bonuses are gained by the creature with specified class/kit with levels.
These files are assigned in KITLIST.2DA to corresponding kit numbers and identifiers.
The first column consists of several identifiers in format ABILITY* (where * is a number on list).
The rest columns consist of added abilities per level. Description of their editing is included beneath.
In IWD2 we have on example of CLABMO01 (for monk):

Give it a file name no longer than the other "SPCL" spells in the game, and add it to the column number equal to the level you want this to be applied at, so it would appear like this
